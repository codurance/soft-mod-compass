# Learn to Debug - Instructions

## Goal
Learn how to implement a Feature in a code base:
- With no test
- With unreadable code

The goal is to be able to:
- Confidently implement the feature
- Making sure that it works
- Improving code quality
- While at the same time ensuring we don't break any existing logic.

**Feature:** The user info returned by the hubspot API is formatted with TitleCase

## Steps
### Preparation
- Remove all tests
- Make code unreadable
- Remove feature

### Exercise
#### Learner
- Implement the feature
- Express clearly the though process

#### Facilitator
- Make sure the correct steps are followed
    - Ensure care is taken to preserve original behavior
    - Ensure new behavior is TDDed
- Help by asking questions
